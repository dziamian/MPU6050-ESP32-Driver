Initializes MPU6050 device instance.

Prepares I2C communication for given port and configuration and wakes the device from the sleep mode. This method need to be called before using any other high-level driver functions.

Parameters
devicePointer to MPU6050 characteristics instance to initialize
bufferPointer to data buffer of the device
bufferSizeSize of the data buffer
portPort where I2C driver will be installed
i2cWaitMsTime allowed to perform the single read/write operation
configConfiguration of the I2C driver
Returns
mpu6050_error_t Status code of the operation
